# Weight & Biases Workshop by Data@UCI

We will learn how to use [Weights & Biases](https://wandb.ai) for experiment tracking and hyperparameter sweeps. You will gain hands-on experience in:
- live loss and accuracy curves
- hyperparameter logging
- artifact versioning
- hyperparameter tuning w/ sweep

## Prerequisites

- Python 3.10+
- A free W&B account: [wandb.ai/signup](https://wandb.ai/signup) (Academic plan available for students)

## Setup

1. Clone the repo
2. Open and run `setup.ipynb`
3. Select the **wandb-demo** kernel
4. Open and run `train.ipynb` to start
4. Log in to W&B when prompted
5. Run all cells and watch your metrics update live at [wandb.ai](https://wandb.ai)
